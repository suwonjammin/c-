프로그램 안으로 파일을 읽어 들이거나 프로그램에서 처리한 데이터를 파일로 내어보내야 할 경우 처리 순서
1.어떤 파일과 데이터를 주고받을지 결정합니다. 이것을 "파일 열기"라고 합니다.
입력으로 할지,출력으로 할지 결정합니다. 이것을 "입출력 모드"라 합니다.
"파일의 주소"를 프로그램 내부에서 알고 있어야 합니다.
입출력 모드가 입력이면 프로그램에서 처리할 데이터를 읽어 옵니다.
입출력 모드가 출력이면 프로그램에서 처리한 데이터를 내어 보냅니다.
입출력 처리가 끝났으면 프로그램은 파일에 놓아야 합니다. 이것을 "파일 닫기"라고 합니다.

모드: w(쓰기),r(읽기),a(추가)
fopen("파일 이름","모드")
fclose(fp)=열엇던 파일을 닫아줌 
fp의 값이 NULL이면 파일 열기에 실패
f(file)put(넣어준다)c(char) 
fgetc(파일 포인터):파일에서 문자를 차례대로 읽어온다. 여러 번 사용하면 다음 문자를 읽어온다.
fputc("문자열",파일 포인터) 문자열을 써준다.
fgetc(배열이름,길이,파일 포인터);
파일에서 문자열 길이 만큼 입력 받아 배열에 저장.
EOF 또는 \n까지 입력 받고, 마지막에 \0을 붙여줌.
stdout:표준 출력 시스템
스트림:프로그램 내/외부로 입출력되는 데이터의 흐름
stdin:표준 입력 스트림(키보드)
stderr:표준 에러 스트림(컴퓨터)
fprintf(파일포인터,"문자열");
파일에 데이터 출력
기본적으로 파일포인터가 맨 앞에 오는 것을 제외하고 printf()와 사용방법이 같다.
stdout에 출력도 가능.
fscanf(파일포인터,"형식문자",변수 주소값);
파일에서 데이터 입력 받음
